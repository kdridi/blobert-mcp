# Contributor Covenant Code of Conduct

This project adopts the **Contributor Covenant v2.1** as its code of conduct.

The full text is available at:
https://www.contributor-covenant.org/version/2/1/code_of_conduct/

## Scope

This code of conduct applies within all project spaces and when an individual
is representing the project in public spaces.

## Enforcement

Instances of unacceptable behavior may be reported by opening an issue on this
repository at https://github.com/kdridi/blobert-mcp/issues or by contacting
the repository maintainers directly.

All reports will be reviewed and investigated promptly and fairly. Project
maintainers are obligated to respect the privacy and security of the reporter.
