# BLO-068: Claude Desktop integration guide

## Objective

Create a step-by-step guide for connecting blobert-mcp to Claude Desktop. The guide should cover MCP server configuration in `claude_desktop_config.json`, ROM file setup, and a first analysis walkthrough with screenshot examples. The guide lives at `docs/guides/claude-desktop.md`, and a quick-start section should be added to `README.md`.

## Context

Claude Desktop supports MCP servers natively, making it a primary integration target for blobert-mcp. Users need clear instructions to configure the connection, load ROM files, and perform their first decompilation analysis. This guide bridges the gap between installation (BLO-066) and productive use.

## Acceptance Criteria

- [ ] Guide created at `docs/guides/claude-desktop.md`
- [ ] Guide covers MCP server configuration in `claude_desktop_config.json` (both pip-installed and Docker variants)
- [ ] Guide includes ROM file setup instructions (where to place files, supported formats)
- [ ] Guide includes a first analysis walkthrough demonstrating key MCP tools
- [ ] Guide includes screenshot examples of Claude Desktop analyzing a ROM
- [ ] `README.md` updated with a quick-start section linking to the full guide
- [ ] All instructions tested and verified to work end-to-end

## Technical Approach

1. Create `docs/guides/` directory if it does not exist.
2. Write `docs/guides/claude-desktop.md`:
   - **Prerequisites**: Python 3.10+, Claude Desktop installed, blobert-mcp installed via pip.
   - **Configuration**: Show exact JSON to add to `claude_desktop_config.json` for both pip and Docker setups:
     ```json
     {
       "mcpServers": {
         "blobert": {
           "command": "blobert-mcp",
           "args": []
         }
       }
     }
     ```
   - **ROM Setup**: Where to place ROM files, supported formats, any size limitations.
   - **First Analysis Walkthrough**: Step-by-step example using the MCP tools (load ROM, read header, disassemble, etc.).
   - **Screenshots**: Include annotated screenshots showing Claude Desktop interacting with the MCP server.
   - **Troubleshooting**: Common issues (server not found, connection errors, ROM format errors).
3. Update `README.md`:
   - Add a "Quick Start with Claude Desktop" section near the top.
   - Link to the full guide for detailed instructions.
