# BLO-010: Update CI actions to Node.js 24 versions

## Objective

Update GitHub Actions in `ci.yml` to versions that run on Node.js 24, eliminating the deprecation warning. Node.js 20 actions will be forced to Node.js 24 on June 2nd, 2026 and removed entirely on September 16th, 2026.

## Context

After BLO-009 landed, the CI workflow triggered a GitHub deprecation warning: actions/checkout@v4, actions/setup-python@v5, and astral-sh/setup-uv@v4 all run on Node.js 20. Updated major versions are available for all three.

## Acceptance Criteria

- [x] `actions/checkout` updated to `v5`
- [x] `actions/setup-python` updated to `v6`
- [x] `astral-sh/setup-uv` updated to `v7` (v5/v6 still on Node.js 20; v7 is the Node.js 24 release)
- [ ] CI workflow runs without Node.js 20 deprecation warning (verify after push)

## Technical Approach

Bump the three action version tags in `.github/workflows/ci.yml`.

## Dependencies

- **BLO-009** — Created the CI workflow.

## Files Modified

- `.github/workflows/ci.yml` — Bumped actions/checkout v4→v5, actions/setup-python v5→v6, astral-sh/setup-uv v4→v7

## Notes

- Minimum runner version for Node.js 24 actions: v2.327.1 (GitHub-hosted runners already satisfy this).
- Reference: https://github.blog/changelog/2025-09-19-deprecation-of-node-20-on-github-actions-runners/
